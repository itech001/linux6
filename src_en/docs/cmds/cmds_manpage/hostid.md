#hostid  



###name
```
       hostid - print the numeric identifier for the current host

```
###synopsis
```
       hostid [OPTION]

```
###description
```
       Print the numeric identifier (in hexadecimal) for the current host.

       --help display this help and exit

       --version
              output version information and exit

```
