#cksum  



###name
```
       cksum - checksum and count the bytes in a file

```
###synopsis
```
       cksum [FILE]...
       cksum [OPTION]

```
###description
```
       Print CRC checksum and byte counts of each FILE.

       --help display this help and exit

       --version
              output version information and exit

```
