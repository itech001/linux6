#link  



###name
```
       link - call the link function to create a link to a file

```
###synopsis
```
       link FILE1 FILE2
       link OPTION

```
###description
```
       Call the link function to create a link named FILE2 to an existing FILE1.

       --help display this help and exit

       --version
              output version information and exit

```
