#unlink  



###name
```
       unlink - call the unlink function to remove the specified file

```
###synopsis
```
       unlink FILE
       unlink OPTION

```
###description
```
       Call the unlink function to remove the specified FILE.

       --help display this help and exit

       --version
              output version information and exit

```
